# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 4.x     | Yes       |
| < 4.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by emailing **luccarebtoledo@gmail.com** instead of opening a public issue.

Please include the following in your report:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact or severity

You can expect an initial response within 72 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.
